Instructions for Use:
    1. Although you do not explicitly need an API key, it may be used to make smarter/faster responses. To use a personal key, you must set its' environment variable which is usually defined on their websites. For example Gemini/Google models use 'GOOGLE_API_KEY' and OpenAI use 'OPENAI_API_KEY' as their monikers.
    2. After setting your environment, be sure to edit config.java for use-case.
